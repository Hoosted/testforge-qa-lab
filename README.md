# TestForge QA Lab

O projeto foi reiniciado como um playground modular para treino de QA.

Toda a fonte de verdade de produto, arquitetura, UI e convencoes agora vive em [AGENTS.md](./AGENTS.md).

## Como rodar

```bash
pnpm install
pnpm dev
```

## Scripts

- `pnpm dev`
- `pnpm build`
- `pnpm test`
- `pnpm typecheck`
- `pnpm e2e`
